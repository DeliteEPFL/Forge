Forge currently works with and requires Lightweight Modular Staging branch delite-develop and Delite branch develop

  1. Download and build both of these repositories on the corresponding branch. Make sure that you run sbt publish-local on each of them (in Delite, only the runtime, framework, and tests projects need to be published). Set the DELITE_HOME environment variable to the root of the Delite repository and the FORGE_HOME environment variable to the root of the Forge repository.

  2. Compile Forge using sbt compile from the root of the Forge repository. The SimpleVectorDSL example can then be run using:

    bin/update ppl.dsl.forge.examples.SimpleVectorDSLRunner SimpleVector
cd published/simplevector/

  3. Run the sample application (apps/simplevector/src/HelloWorld.scala)

    Interpreter (library mode):

    bin/delitec HelloSimpleInterpreter

    Compiler (Delite mode):

    bin/delitec HelloSimpleCompiler
bin/delite HelloSimpleCompiler
